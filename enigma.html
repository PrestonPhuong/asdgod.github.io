How to Use the Enigma Machine

  • Set the rotor types (I-V) for each of the three rotors.
  • Set the initial rotor positions (e.g., "ABC") or use the default "AAA".
  • Configure the plugboard by entering letter pairs (e.g., "AB CD EF").
  • Type your message in the Input Text area (only A-Z letters are processed).
  • Click ENCRYPT / DECRYPT to process the text.
  • Each letter you type will light up on the lightboard showing the encrypted result.
  • The Enigma machine is symmetric: encrypting the ciphertext with the same settings returns the original message.

Note: This simulation includes the standard Enigma rotor stepping mechanism where the rightmost rotor advances with each keypress, and other rotors step when certain positions are reached.
